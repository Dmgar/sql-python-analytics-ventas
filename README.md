# Análisis de Ventas con Python y SQLite

Este repositorio contiene un análisis completo de datos de ventas utilizando **Python** y **SQLite**. El proyecto demuestra cómo realizar consultas SQL desde Python para extraer KPIs, métricas de negocio y análisis exploratorio de datos.

## Contenido del Proyecto

El notebook `Taller_consultas.ipynb` incluye las siguientes consultas analíticas:

### KPIs Principales
- Total de ingresos, costos y utilidad general

### Análisis Temporal
- Ingresos totales por año y mes
- Crecimiento mensual de ingresos (2024-2025)

### Análisis por Dimensiones
- Utilidad por canal de ventas (Online vs Offline)
- Ventas por región geográfica

### Análisis de Productos
- Top 5 productos con mayor margen de ganancia
- Productos con margen bajo (<20%)
- Porcentaje de participación de cada producto sobre ventas totales

### Análisis de Clientes
- Gasto promedio por compra

### Análisis Estacional
- Días de la semana con mayores ventas (año 2025)
